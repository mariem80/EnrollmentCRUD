A C#/.NET Web API for managing student enrollments in courses using CRUD operations for students and enrollments.

**Technologies & Patterns Used**
- Clean Architecture 
- ASP.NET Core Web API (RESTful Services)
- Entity Framework Core (ORM for Database Operations)
- Repository & Unit of Work Patterns
- Custom Middleware (Global Exception Handling)
- Generic API Response Wrapper
- AutoMapper for Mapping
- FluentValidation (Request Validation)
- CQRS with MediatR (Command/Query Separation)
- .resx Resource File (Localized Validation & Error Messages)
